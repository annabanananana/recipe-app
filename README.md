# recipe-app

## Project setup
```
git clone https://github.com/annabanananana/recipe-app.git
cd recipe-app/backend
npm install
cd recipe-app
npm install
```
#### run the front-end app
```
cd recipe-app
npm run serve
```
##### run the back end
```
cd recipe-app/backend
node index.js
```
